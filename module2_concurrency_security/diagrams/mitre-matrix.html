T1553.002

T1553.002 · Subvert Trust Controls: Code Signing
Adversary subverts code signing to disguise malicious code.
npm provenance creates valid attestations for malicious packages because the package WAS legitimately built from a commit in the right repo. Provenance proves origin, not safety. A reviewer seeing "attestation: valid" gets false confidence. This is provenance-as-safety-theater.
